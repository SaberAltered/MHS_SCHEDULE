# My School Schedule & Event Tracker

My own little calander designed for mobile and pc that shows what im doing through the day and can be used for your own events too. This Mini project automatically tracks Whether its a **A Or B Day** using the MHS Calander and automatically tracks whether theres an off day or a holiday,

## Check it out here

 [https://saberaltered.github.io/MHS_SCHEDULE/](https://saberaltered.github.io/MHS_SCHEDULE/)

---

## ✨ Key Features
* **A/B days based on the school calendar**
* **Real-Time Sync**
* **Custom Themes:** A settings menu in the bottom left to change background images and add local events.
* **Live Clock:** 12-hour digital clock that highlights what class im in

---

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
* **Logic:** Vanilla JavaScript (Async/Fetch API)
* **Hosting:** GitHub Pages
* **Data:** JSON (Serverless)
